# GabungGambar

Static SEO tool site for `gabunggambar.org`.

## Pages

- `/` - main landing page for `gabung file gambar`
- `/gambar-ke-pdf/`
- `/gabung-foto-jadi-pdf/`
- `/jpg-ke-pdf/`
- `/png-ke-pdf/`

## Deploy

1. Push this directory to GitHub.
2. Import the repo into Vercel as a static site.
3. Point DNS to Vercel via Cloudflare.
4. Submit `https://gabunggambar.org/sitemap.xml` to Google Search Console.
