Kebijakan Privasi

GabungGambar dirancang untuk memproses gambar langsung di browser. Artinya file yang Anda pilih diproses di perangkat Anda selama sesi penggunaan tool.

  • Kami tidak meminta akun untuk memakai fitur utama gambar ke PDF.
  • File yang dipilih digunakan hanya untuk membuat PDF dalam sesi browser aktif.
  • Jika analitik ditambahkan di masa depan, kebijakan ini akan diperbarui agar transparan.
